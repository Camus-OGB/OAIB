# OAIB Frontend

## Installation

```bash
npm install
```

## Configuration Supabase

1. Créez un projet Supabase et récupérez l'URL du projet ainsi que la clé `anon`.
2. Copiez le fichier `.env.example` en `.env` et renseignez :
	- `VITE_SUPABASE_URL`
	- `VITE_SUPABASE_ANON_KEY`
3. Rechargez le serveur Vite après toute modification des variables.

## Scripts utiles

- `npm run dev` : démarre le serveur de développement.
- `npm run build` : génère la build de production.
- `npm run preview` : prévisualise la build.
