# OAIB Frontend

Site vitrine des Olympiades d'Intelligence Artificielle du Bénin.

## Installation

```bash
npm install
```

## Scripts utiles

- `npm run dev` : démarre le serveur de développement.
- `npm run build` : génère la build de production.
- `npm run preview` : prévisualise la build.

## Structure du projet

- `/src/public` : Pages publiques du site vitrine
- `/src/admin` : Espace d'administration
- `/src/student` : Espace candidat/étudiant
- `/src/features/auth` : Pages d'authentification
- `/src/shared` : Composants et utilitaires partagés
